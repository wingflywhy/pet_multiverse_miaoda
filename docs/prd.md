# My Pet's Multiverse需求文档

## 1. 应用概述

### 1.1 应用名称
My Pet's Multiverse

### 1.2 应用描述
一个结合AI图片生成与编辑的创意应用，为用户的宠物（或虚拟宠物）创造出荒诞、有趣且高度定制化的平行宇宙形象，激发用户的社交分享欲望。

### 1.3 目标用户
18-35岁的年轻人，喜爱宠物、热衷社交分享、乐于尝试新鲜事物和AIGC工具。

## 2. 核心功能

### 2.1 宇宙生成器

#### 2.1.1 宠物信息输入
用户输入或选择宠物的种类（如：柴犬、橘猫）

#### 2.1.2 平行宇宙设定
用户选择或输入宠物的「离谱」新身份和任务场景

#### 2.1.3 风格选择
用户选择最终图像的艺术风格（如：写实、卡通、油画、赛博朋克等）

#### 2.1.4 指令生成\n系统根据用户输入，自动生成优化、细节丰富的Nano生图指令

#### 2.1.5 图像生成
调用Nano生图API，根据生成的指令产生最终图像

### 2.2 辅助功能

#### 2.2.1 图库展示
展示用户已生成的历史作品

#### 2.2.2 一键分享
支持将生成图片一键分享至社交平台，并自动带上产品水印和Slogan

#### 2.2.3 创意模板库
预设一些热门且有趣的「身份+场景」组合，供用户快速选择

## 3. 用户操作流程

### 3.1 步骤1：进入首页
用户点击「开始创造宠物宇宙」按钮
\n### 3.2 步骤2：宠物输入
必填项：宠物种类（如：柴犬、橘猫）
选填项：上传宠物照片（未来扩展）

### 3.3 步骤3：设定宇宙
必填项：
- ① 新身份（下拉选择或自由输入）
- ② 任务场景（下拉选择或自由输入）

### 3.4 步骤4：选择风格
选择一种艺术风格

### 3.5 步骤5：生成指令
系统在后台自动优化并生成详细的AIGC指令，并显示「指令生成中...」

### 3.6 步骤6：等待生成
屏幕显示加载动画和「正在探索您的宠物宇宙，预计等待 XX 秒」

### 3.7 步骤7：结果展示
展示生成的最终图像，提供「下载」「分享」「再试一次」按钮

## 4. AI生图指令生成逻辑

### 4.1 输入变量
- Pet_Type：用户输入的宠物种类（如：柴犬）
- New_Identity：用户输入的新身份（如：国际特工）
- Scenario：用户输入的任务场景（如：雪山顶上秘密接头）
- Style：用户选择的艺术风格（如：写实）

### 4.2 指令结构
最终的Nano生图指令P遵循「主体 + 细节/动作 + 场景/背景 + 风格/质量」的结构：

P = (主体描述) + (动作与服装细节) + (场景环境细节) + (风格与质量关键词)

### 4.3 自动生成规则

#### 主体描述\n`一只拟人化，穿着全套服装的 {Pet_Type}`

#### 动作与服装
结合New_Identity自动添加（例：身份是「特工」则添加：「身穿定制黑色西装，戴着墨镜，表情严肃，手里拿着对讲机」）

#### 场景环境
结合Scenario自动添加环境和气氛（例：场景是「雪山顶上」则添加：「白雪皑皑的山顶，远处一架直升机螺旋桨正在旋转，雪花飞舞，营造紧张而神秘的氛围」）

#### 风格与质量
`{Style}，超高清，8K，电影级照明，细节丰富`

### 4.4 示例指令
基于用户示例（柴犬+国际特工+雪山顶+写实）：

> 一张写实风格的照片。一只拟人化的柴犬，身穿定制黑色西装，戴着墨镜，表情严肃地站在白雪皑皑的山顶上。它的一只爪子搭在对讲机上，远处一架直升机螺旋桨正在旋转，雪花在风中飞舞。背景是连绵起伏的雪山和灰蒙蒙的天空，营造出紧张而神秘的氛围，超高清，8K，电影级照明，细节丰富。

## 5. 技术要求

- 系统稳定接受用户输入并成功转化为结构化生图指令
- Nano生图API调用成功率≥95%
- 生成图像与用户输入的「宠物」「身份」「场景」「风格」高度匹配
- 图像质量清晰、细节丰富，具有艺术性和传播性
- 分享功能能正确生成带水印图片并引导社交分享

## 6. 设计风格

- **配色方案**：以温暖的橙色为主色调，搭配深蓝色和白色，营造创意且专业的科技感
- **视觉细节**：采用圆润的8px圆角设计，柔和的渐变阴影效果，线性图标风格，流畅的加载动画
- **整体布局**：卡片式布局，内容层次分明，重点突出生图结果区域，确保用户体验流畅自然
